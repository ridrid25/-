# Калькулятор АБС — себестоимость и маржинальность асфальтобетона

Мобильный B2B-калькулятор себестоимости, маржи и скидок для асфальтобетонных
смесей. React + TypeScript + Tailwind + lucide-react. Светлая и тёмная темы,
роли менеджер/админ, карта торга, журнал расчётов и счёт в Telegram.

Оформление — в фирменной палитре **«Нефть-зелёный» (Petrol)**.

## Возможности

- **Ассортимент марок** — сворачиваемые карточки с расчётом себестоимости,
  цены, маржи и карты торга по каждой марке.
- **Карта торга** — наглядная шкала зелёная/жёлтая/красная зона со скользящим
  маркером: до какой скидки держится целевая маржа и где начинается убыток.
- **Движок (bottom sheet)** — быстрый подбор скидки ползунком, ключевые метрики
  и (для админа) оперативная правка цен сырья по марке.
- **Параметры модели** — НДС, целевая маржа, логистика (самовывоз/доставка
  рейсами), цены сырья по группам, производственные расходы на тонну.
- **Журнал расчётов** — буфер из 7 последних расчётов с залипающей колонкой
  «Марка» и кнопкой формирования счёта в Telegram.
- **Роли** — менеджер видит и торгуется скидкой; админ (по PIN) меняет
  закупочные цены и расходы, сохраняет их в браузере.
- **Справочник рецептур** — нормы расхода кг/т по каждой марке (из документов).
- **Тёмная тема** — переключение классом `.dark`, выбор запоминается.

## Структура

| Файл | Назначение |
|---|---|
| `src/CalculatorEngine.ts` | Вся бизнес-логика: типы, каталог сырья, рецептуры, `compute()` (себестоимость, НДС, карта торга, точка безубыточности), текст счёта для Telegram. **UI не содержит.** |
| `src/CalculatorCard.tsx` | Карточка марки: поля ввода, метрики, карта торга, кнопки «Движок» / «Сохранить». |
| `src/HistoryTable.tsx` | Журнал расчётов: таблица с залипающей колонкой и счётом в Telegram. |
| `src/App.tsx` | Обёртка: тема, роли/PIN, настройки, движок (bottom sheet), справочник, модалки, тосты. |
| `src/NumberInput.tsx` | Числовое поле, сохраняющее промежуточный ввод (`1.`, `-`, `-1.5`). |
| `src/Flash.tsx` | Короткая подсветка значения при изменении. |
| `src/index.css` | Палитра (CSS-переменные), шрифты, переходы, анимации, ползунок. |
| `tailwind.config.js` | `darkMode: 'class'`, токены палитры и шрифты. |

## Установка и запуск

```bash
npm install
npm run dev        # дев-сервер Vite
npm run build      # сборка в dist/ (tsc + vite build)
npm run preview    # предпросмотр собранной версии
npm run lint       # проверка типов (tsc --noEmit)
```

Зависимости рантайма: `react`, `react-dom`, `lucide-react`. Веб-шрифты
подключены в `index.html` через Google Fonts.

## Дизайн-система «Нефть-зелёный»

Единый источник цвета — CSS-переменные в `src/index.css` (хранятся как
триплеты `R G B`, чтобы работали модификаторы прозрачности Tailwind вида
`bg-accent/40`). Токены проброшены в `tailwind.config.js` и автоматически
переключаются под классом `.dark`.

| Токен | Светлая | Тёмная |
|---|---|---|
| `bg` — фон страницы | `#F0F3F2` | `#0A1817` |
| `surface` — карточки/панели | `#FFFFFF` | `#11302F` |
| `line` — границы | `#D3DAD8` | `#234240` |
| `ink` — основной текст | `#0F3D3E` | `#E2EEEC` |
| `muted` — второстепенный текст | `#5E7370` | `#8FAAA6` |
| `accent` — интерактивный акцент | `#0F3D3E` | `#C2703D` (медь) |
| `brand` — шапка/хром | `#0F3D3E` | `#0F3D3E` (петроль) |
| `copper` — хайлайт | `#C2703D` | `#C2703D` |

- **Медь `#C2703D`** — цены со скидкой, маркер карты торга, главные CTA и
  детали-акценты.
- **Бренд-хром** (шапка, шапка справочника, бейджи секций, шапка журнала)
  остаётся петролем в обеих темах; в тёмной теме медь работает только как
  деталь-акцент.
- **Смысловые цвета зон маржи** не меняются (несут смысл): зелёный `#2F9E63`,
  жёлтый `#E0A93A`, красный `#CF4B4B` (в тёмной теме чуть высветлены).

### Шрифты

- **Space Grotesk** — заголовки и шапка.
- **Inter** — основной текст и подписи.
- **IBM Plex Mono** + `tabular-nums` — все денежные значения, проценты, объёмы,
  журнал и движок (цифры выровнены в колонки и не «прыгают»).

### Тактильность

Сдержанные, быстрые детали: плавные переходы (`cubic-bezier(0.4,0,0.2,1)`),
поднятие кнопок на hover и «вдавливание» на нажатии, кольца `focus-visible`,
медное свечение главных CTA, крупный медный бегунок ползунка, плавное
раскрытие карточек, скользящий маркер карты торга, «дыхание» красной
зоны-точки, подсветка изменившихся чисел. Уважается `prefers-reduced-motion`.

## Настройки (PIN, Telegram, компания)

В начале `src/CalculatorEngine.ts` — объект `CONFIG`:

```ts
export const CONFIG = {
  ADMIN_PIN: '2026',          // PIN администратора (простая защита от случайных правок)
  TG_CHAT_ID: '300023851',    // chat ID бухгалтера
  TG_PHONE: '+79183033323',   // телефон бухгалтера
  COMPANY: 'ООО «Фирма ЮДС»', // шапка счёта
};
```

## Важные оговорки

- **PIN в коде** защищает только от случайных правок (его видно в исходнике).
  Настоящая защита цен — через бэкенд.
- **Счёт в Telegram**: формируется готовый текст, копируется в буфер и
  открывается чат с бухгалтером. Автоматическая отправка ботом требует своего
  сервера (токен бота нельзя держать в коде страницы). Место помечено в
  `App.tsx` (`sendInvoice`).
- **НДС 22%** по ТЗ (в РФ обычно 20%) — константа `VAT_RATE` в движке.
- **Сохранение** цен/расходов — пока в `localStorage` (только админ). Легко
  перевести на сервер (`loadSettings` / `saveData`).
- **Фото рецептов**: оригиналы документов можно положить в
  `public/recipes/<key>.jpg` и подставить в справочник (см. комментарий в
  `RefDrawer` в `App.tsx`).
